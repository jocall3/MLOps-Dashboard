# MLOps Dashboard

A comprehensive predictive models hub featuring real-time monitoring, drift detection, explainability (XAI), and Gemini-powered insights for machine learning operations.

## Features
- Model Registry & Management
- Performance Monitoring & Analytics
- Automated Deployment (Blue/Green, Canary)
- Explainable AI (SHAP/LIME visualization)
- Feature Store Integration
- Deep File Analysis using Google Gemini 3 Pro
